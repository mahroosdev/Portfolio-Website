# Dice Duel

A simple two-player dice game made for **Boss Level Challenge 01 – The Dice Game**.

## Features

- Two randomly generated dice rolls
- Winner or draw result after every round
- Responsive layout for mobile and desktop
- No libraries or installation required

## How to run

1. Download or clone this project.
2. Open `index.html` in a web browser.
3. Click **Roll Dice** to play.

## Files

- `index.html` – page structure
- `style.css` – design and responsive layout
- `script.js` – dice rolling game logic

## Built with

HTML, CSS and JavaScript.
