# Vertex IT Solutions

A responsive single-page IT solutions website created with HTML, CSS, and Bootstrap 5.

## Included sections

- Responsive navigation bar
- Hero section and call-to-action buttons
- Services cards
- About and approach section
- Why choose us section
- Contact call-to-action and footer

## How to run

Open `index.html` in any modern web browser. An internet connection is required for the Bootstrap and Google Fonts CDN links.

## Technologies used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Notes

This is a frontend UI project. The contact email is a placeholder and does not submit data.
