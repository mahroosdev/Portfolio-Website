# Simon Says — Memory Game

A responsive browser-based Simon game built with HTML, CSS, and JavaScript.

## How to play

1. Click **Start Game** (or press Enter/Space).
2. Watch the coloured button pattern.
3. Repeat the pattern in the same order.
4. Every successful round adds one more colour.

Your highest score is saved in the browser using local storage.

## Run locally

Open `index.html` in a browser. No installation or server is required.

## Files

- `index.html` — game structure
- `style.css` — responsive visual design
- `script.js` — Simon gameplay and sound effects
